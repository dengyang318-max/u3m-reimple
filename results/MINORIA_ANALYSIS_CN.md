# College Admission Official-Utils — Minoria 特征总结（结合百分位表与可视化）

基于 `ALL_RUNS_GRIDS_GROUPED.md` 的 top1/2/3 百分位表与九宫格可视化（full / tail / density）进行总结。

## 核心结论
- **vector_transfer 开/关** 决定性地改变发现的尾部与少数群体特征；关闭后在中等尾部出现更多正例，F1 明显抬升，并伴随更大的 F/M 比例波动。
- **归一化 L1 vs. L2** 仅带来细微漂移，少数群体构成基本不变。
- **其他开关（linkedlist、first_intersection_init、min_shift）** 仅引起极小排序/偏移差异，不改变少数群体本质。
- **极小百分位 (q≤0.04)** 各组普遍正例稀疏，F1→0，尾部更多体现“几何稀疏”而非“标签富集”。

## 分组对比与图表解读

### A 组（基线：vector_transfer 开，L1）
- **百分位表特征**：在 q=0.10–0.04，F1 常为 0，Accuracy 高（~0.77–0.88），说明尾部极少正例，偏负；F/M 比例在 1–1.8，性别偏移轻微。  
- **可视化**：tail / density 图显示尾部点云稀疏且分散；full 图分布均匀，无明显偏斜集中。  
- **结论**：少数群体主要由“稀疏方向”定义，并非标签富集；人口学偏移弱。
- **示例图片**：`20251208_205408/official_utils_top_grid.png`（3×3：行 full/tail/density，列 top1/2/3）；A 组各 run 形态相似。  
- **百分位范围**：q = 1.00, 0.50, 0.20, 0.10, 0.08, 0.04（top1/2/3 表一致）。

### B 组（vector_transfer 关闭，变化最大）
- **百分位表特征**：中等尾部 (q≈0.50–0.10) F1 显著高于 A 组（如部分 run top-1 F1≈0.56–0.57），说明尾部保留了更多正例；极小尾部 q=0.04 仍可能 F1=0。F/M 比例波动更大（可>1.4 或 <1），人口学偏移更明显。  
- **可视化**：tail / density 图中，尾部区域点更集中，正例（颜色）占比更高；full 图中的方向选择与 A 组不同，反映了未做 vector_transfer 时找到的“原空间”方向。  
- **结论**：关闭 vector_transfer 能找到“标签更强、偏移更显著”的少数群体，是唯一显著改变群体构成的开关。
- **示例图片**：`20251208_205502/official_utils_top_grid.png`（尾部正例更集中），`20251208_205636/official_utils_top_grid.png`（相似趋势）。  
- **百分位范围**：q = 1.00, 0.50, 0.20, 0.10, 0.08, 0.04（top1/2/3 表一致）。

### C 组（L2 归一化，vector_transfer 开）
- **百分位表特征**：与 A 组相近，F1 在紧尾部仍多为 0，指标轻微漂移；F/M 比例无明显放大。  
- **可视化**：tail / density 图形态与 A 组接近，尾部稀疏，未出现新的集中簇。  
- **结论**：L2 仅带来数值级微调，少数群体与 A 组基本一致。
- **示例图片**：`20251208_205710/official_utils_top_grid.png`（代表性 L2 结果，形态近似 A 组）。  
- **百分位范围**：q = 1.00, 0.50, 0.20, 0.10, 0.08, 0.04（top1/2/3 表一致）。

## 参数对少数群体的影响
- **最关键**：`disable_vector_transfer`  
  - 关：中尾部正例增多，F1 抬升，F/M 波动更大。  
  - 开：尾部更稀疏、偏负，F1 易归零，人口学偏移小。
- **次要**：`use_l2_norm`（仅细微漂移）。  
- **可忽略**：`use_linkedlist`、`use_first_intersection_init`、`disable_min_shift`（只影响顺序/微调）。

## 观察与使用建议
- 若要稳定复现“官方风格”尾部：保持 vector_transfer 开、L1；预期尾部正例稀少，F1 低，人口学偏移轻。
- 若要挖掘标签更强、偏移更明显的少数群体：比较 B 组（vector_transfer 关）与 A 组。
- 查看具体 run 时，可结合九宫格：full 判断方向与整体分布，tail 与 density 观察尾部点的集中度与正例占比。
