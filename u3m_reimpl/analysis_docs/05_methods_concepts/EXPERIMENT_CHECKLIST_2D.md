## 实验汇总功能说明

新增的`print_experiment_summary`函数会输出：
1. **数据集信息**：点数
2. **性能指标**：各策略的运行时间，动态vs固定的加速比
3. **方向对比**：最佳偏度值、方向向量、Top-3偏度范围
4. **策略对比**：固定vs动态中位数的偏度差异和方向差异（L2范数）
5. **旋转点集结果**：旋转点集的最佳偏度方向

这个汇总在实验结束时自动输出，方便快速查看实验结果。
